# The National Energy Modelling Framework

This is a highly spatially resolved process heat tool designed to identify fuel-switching options at a systems level.

It is fully coded in Python. The framework is open source, although licences may be required for some imported libraries.

It has been tested using a New Zealand case study.

Full documentation and the New Zealand case study will be available soon.


The simple documentation are as follows:

### E1 - a modified version of the P-graph Python library 
This essentially helps with the data curation and data extraction process.

### E2 - a P-graph Pyomo porting tool
The translation tool converts P-graph input text file into Pyomo-ready problem

## E3 - Pre-processing tools
Includes data generation and data extraction from server. Exogenous to the modelling framework.

## E4 - Output layer
Python output files which include masks that generate tables from raw results

## E5 - Case Study codes
Includes three simulation modes
